Quick Start

Run Lamb from a terminal

Lamb only needs a JDK. No Gradle, Maven, or package manager is required. 

$build = "out"
New-Item -ItemType Directory -Force -Path $build | Out-Null
javac -d $build (Get-ChildItem -Recurse -Filter *.java | Select-Object -ExpandProperty FullName)
mkdir -p out
javac -d out $(find src -name "*.java")

Start the REPL

java -cp out com.lambinterpreter.lamb.Lamb

Run a file

java -cp out com.lambinterpreter.lamb.Lamb path/to/program.lamb

Regenerate AST sources

java -cp out com.lambinterpreter.tool.GenerateAst src/com/lambinterpreter/lamb

Control Flow

Branch and loop through code

If and else

var score = 90;

if (score >= 50) {
    print "pass";
} else {
    print "try again";
}

While

var n = 3;

while (n > 0) {
    print n;
    n = n - 1;
}

For

for (var i = 0; i < 3; i = i + 1) {
    print i;
}

Logical operators

if (true and false) {
    print "won't run";
}

if (nil or "fallback") {
    print "fallback wins";
}

Loop Control

Break and continue

Use break to exit a loop early, and continue to skip to the next iteration. Both are validated by the resolver — using them outside a loop is a compile-time error.

Break

for (var i = 0; i < 10; i = i + 1) {
    if (i == 5) break;
    print i;
}
// Prints 0 through 4

Continue

for (var i = 0; i < 6; i = i + 1) {
    if (i == 3) continue;
    print i;
}
// Prints 0, 1, 2, 4, 5

Functions

First-class functions and closures

fun makeCounter() {
    var count = 0;

    fun next() {
        count = count + 1;
        return count;
    }

    return next;
}

var counter = makeCounter();
print counter();
print counter();

Function calls check argument count before execution. Parameter and argument lists are capped at 255 entries.

Classes

Objects, methods, fields, and inheritance

class Person {
    init(name) {
        this.name = name;
    }

    introduce() {
        print "hello " + this.name;
    }
}

class Developer < Person {
    introduce() {
        super.introduce();
        print this.name + " writes Lamb";
    }
}

var dev = Developer("Ada");
dev.introduce();
Class rules: init initializes a new instance, methods bind this, and subclasses can reach overridden behavior with super.method().
